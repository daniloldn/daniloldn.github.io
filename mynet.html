MyNet Project
Neural Network Package
Built from Scratch

Project Overview

MyNet is a basic neural network library built entirely from scratch in Python. The package is completed and operational, although further work will need to be done if it is to be benchmarked against other popular neural network packages

Key Features

  • Core Neural Network Components: Forward propagation, backpropagation, and stochastic gradient descent algorithms implemented from first principles
  • Multiple Layer Types: Dense/fully connected layers, activation function: Sigmoid
  • Loss Functions: Mean Squared Error
  • Optimization Algorithms: SGD configurable hyperparameters
  • Model Architecture: Flexible network construction with easy layer stacking and configuration
  • Training Pipeline: Built-in training loops with validation and progress tracking

Learning Outcomes

This project reinforced deep understanding of:

  • Linear algebra applications in machine learning
  • Calculus and optimization theory in practice
  • Software engineering principles for scientific computing
  • The mathematical foundations underlying modern deep learning frameworks
  • Package deployment and Documentation
