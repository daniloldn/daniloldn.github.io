5D Interpolator Web App

Project Overview

I built this full-stack web app to make 5-dimensional dataset interpolation accessible through a clean web interface. The project combines a FastAPI backend with a Next.js frontend, using scikit-learn's neural networks to handle complex interpolation tasks that would be difficult with traditional methods.

How It Works

Backend: FastAPI server running Python 3.11+ with scikit-learn's MLPRegressor

Frontend: Next.js 16 with React 19 and TypeScript for the user interface

Deployment: Dockerized with docker-compose for easy setup and scaling

Data Processing: Automated preprocessing with StandardScaler and train/test splitting

What You Can Do

  • Upload Your Data: Drop in .pkl files - the system validates they're 5-dimensional
  • Configure the Network: Adjust layers, activation functions, and training parameters
  • Watch It Train: Real-time progress updates during model training
  • Get Predictions: Input new 5D points and see interpolated results
  • See Performance: Visual feedback on how well the model is doing

Under the Hood

The backend handles five main operations: file upload with validation, status checking, model training, prediction generation, and cleanup. I built a custom ML package that orchestrates the data flow - from preprocessing your uploaded pickle files (expecting {"X": features, "y": targets} format) through StandardScaler normalization to training the neural network.

The frontend gives you a single-page interface to control everything. Upload your data, tweak the neural network settings, start training, and watch the results come in. The whole thing runs asynchronously so you're not stuck waiting for long training jobs.
