5D Interpolator Web App

Overview

A full-stack web application that provides neural network-based interpolation for 5-dimensional numerical datasets. Upload your data, configure a neural network, and generate predictions through an intuitive web interface.

Key Features

  • Data Upload: Accept pickle files containing 5-dimensional datasets
  • Neural Network Training: Configurable MLPRegressor with customizable architecture
  • Real-time Predictions: Generate interpolated values for new 5D input vectors
  • Interactive UI: Modern React interface with real-time training feedback
  • Dockerized Deployment: Complete containerized setup for easy deployment

Architecture

Backend (FastAPI + Python): FastAPI web framework with async support, scikit-learn MLPRegressor for neural network training, automatic data validation and preprocessing, and secure file management with content-based deduplication.

Frontend (Next.js + React): Next.js 16 with React 19 and TypeScript, Tailwind CSS for responsive design, real-time updates during training and prediction, plus form validation and error handling.

How to Use

Data Preparation: Prepare your dataset as a Python pickle file containing a dictionary with "X" (features as N x 5 array) and "y" (targets as N array).

Training Process: Upload your .pkl file, configure neural network parameters (hidden layers, activation function, optimizer, max iterations), train the model with real-time progress monitoring, and generate predictions by entering 5 comma-separated values.

Technical Details

The data processing pipeline includes validation for 5-dimensional structure, StandardScaler normalization with missing value imputation, 80/20 train/test split, and MLPRegressor training with configurable hyperparameters. The system supports only dictionaries in pickle format for now.
